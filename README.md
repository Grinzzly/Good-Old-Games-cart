## Installation & Usage

From the root of the project directory:

### dev
```
npm i
ng serve
then visit 'http://localhost:4200/'
```

### prod
```
npm i
ng build
then head to 'dist/gog/index.html' from project root directory
```

_Use at least 8.9.0 version of Node._
